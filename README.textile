h1. Usage

Clone this repo into your home directory either as .vim (linux) or vimfiles (MF
Windows).

<pre><code>
$ cd your_path
your_path$ git clone git@github.com:scorpio/vimfiles.git
</code></pre>

Then cd into the repo and run this to get the snippets submodule:

<pre><code>
your_path$ cd vimfiles
your_path/vimfiles$ git submodule init
your_path/vimfiles$ git submodule update
</code></pre>

<pre><code>
ln -sf your_path/vimfiles ~/.vim
ln -sf your_path/vimfiles/vimrc.leon ~/.vimrc
</code></pre>
